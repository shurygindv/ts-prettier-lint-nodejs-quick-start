## Quick start: nodejs + typescript + eslint + prettier

*Updated:* 6/23/2020

Apply changes in your vscode config:
```json
    "prettier.eslintIntegration": true,
    "eslint.validate": [
        "javascript",
        "javascriptreact",
         { "language": "typescript", "autoFix": true },
         { "language": "typescriptreact", "autoFix": true }
    ],
   ```
